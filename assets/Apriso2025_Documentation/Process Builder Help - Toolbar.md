# Toolbar

The toolbar offers convenient shortcuts for executing common tasks. They can be floated or locked
to the application window.

To Move the Toolbar

Drag the (Move Handle) on a toolbar when it is docked and the Lock the Toolbars option is off. If
the toolbar is dropped on the top, bottom, left, or right edges of the application window, it docks
there. If it is dropped anywhere else, it becomes a floating toolbar.
