# HTML Editor

View HTML: HTML Editor

The HTML Editor is a simplified HTML Layout Editor. The editor consists of three tabs: HTML,
JavaScript, and CSS. It supports Inputs and Outputs, Screen Literals and Conditionals.

HTML Tag Attributes

HTML Editor supports special attributes listed here.

HTML Editor also supports additional attributes that can be used in an HTML tag:

           Action
           Button
           Disable Button
           Tab
           Tab Text

These attributes are parsed and interpreted by various converters that generate regular HTML and
JavaScript. You can add your own custom converters that parse custom attributes.

HTML Editor does not support all the attributes that are available in the HTML Layout Editor,
like Controls Tags.

  Due to a third-party issue, the DataList control is not displayed in PB Test Run and in DELMIA
  Apriso Desktop Client. However, it works correctly in DELMIA Apriso Portal.

Toolbox

The Toolbox for the HTML Editor contains custom snippets that can be dragged and dropped to the
HTML tab.
