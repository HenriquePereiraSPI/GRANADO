# View Overview

A View is a user interface element that is displayed inside the Panels on a Screen and is used to
collect or display information from the user.

A View can contain the actual View, an DFC of the View subtype, and Business Controls.

The available View types are:

           Header
           Form
           General

                      Grid View Template
                      Tree View Template
                      Visualization View Template
                      Tab View Template

Creating a View

Copying and pasting Views Templates instead of creating them from scratch is recommended.

A View can be linked to a Screen in the following ways:

           Directly to a Panel
                      A View linked to a Panel on a Screen can be refreshed asynchronously by a given
                      time interval (for details, see AJAX View Template).

           As a Change View
           As a Pop-up Window (via an Action)
