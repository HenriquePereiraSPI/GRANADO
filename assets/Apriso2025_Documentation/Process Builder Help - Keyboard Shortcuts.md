# Keyboard Shortcuts

The shortcuts available in Process Builder can help experienced users save time when working
with the application.

Some shortcuts may not be active on certain screens or in certain views.

Shortcut           Command

CTRL+E             Open Entity Manager. If the Entity Manager is already opened, the Entity
                   Manager and Categories filters are cleared.

CTRL+L             Open the Designer view.

CTRL+T             Open the Test Run view.

CTRL+SHIFT+P Change the entity status to Prototype.

CTRL+SHIFT+G Change the entity status to Development in Progress.

CTRL+SHIFT+D Change the entity status to Design In Progress.

CTRL+ALT+A         Change the entity status to Active recursively.

CTRL+ALT+P         Change the entity status to Prototype recursively.

CTRL+ALT+G         Change the entity status to Development in Progress recursively.

CTRL+SHIFT+B Compile/Run the entity.

CTRL+SHIFT+V Validate the entity.

CTRL+Y             Redo.

CTRL+Z             Undo.

CTRL+C             Copy the selected entity.

CTRL+X             Cut the selected entity.

CTRL+V             Paste the content of the clipboard.

CTRL+F or F3 Find an Entity

CTRL+J             Open the Background Tasks pane.

DEL                Delete the selected item/entity.

SHIFT+F4           Open the Properties window.

F1                 Open the context-sensitive help.

F4                 Open the Properties tool pane.

F5                 Refresh the current view.

F11                Enable/disable the Full Screen view.

CTRL+TAB           Switch between tool panes or tabs.

CTRL+F4 CTRL+W Close the window/tab.

CTRL+N             Launch the New Entity Wizard.

CTRL+O             Open an Entity.

CTRL+S             Save an entity.

CTRL+P             Print.

Keyboard Shortcuts on Step Navigation

Shortcut           Command
                   Print the diagram.
CTRL+P

CTRL+C             Copy the selected Steps to the clipboard.

CTRL+X             Cut the selected Steps.

CTRL+V             Paste Steps from the clipboard (in an external application such as Microsoft
                   Word or Microsoft Paint, they will be pasted as an image).

INS CTRL+INS F7 Add a Step.

CTRL+A             Select all Steps.

CTRL+D             Duplicate the selected Steps.

CTRL+T             Open the Test Run view.

F2                 Rename the selected Step or change the Routing Value of the selected arrow.

SHIFT+F2           Edit the selected Step description.

F4                 Open the Properties tool pane.

F5                 Refresh the current view.

F11                Enable/disable the Full Screen view.

F12                Save the diagram as an image.

DEL                Remove the selected items (if an arrow is selected, the routing value is also

                   removed).

CTRL+scroll        Zoom in/Zoom out.

Keyboard Shortcuts on Function Navigation

Shortcut           Command

CTRL+C             Copy the selected Functions to the clipboard.

CTRL+X             Cut the selected Functions.

CTRL+V             Paste Functions from the clipboard (in an external application such as Microsoft
                   Word or Microsoft Paint, they will be pasted as an image).

CTRL+A             Select all Functions.

CTRL+T             Open the Test Run view.

CTRL+SHIFT+I Add an Input to the selected Function.

CTRL+SHIFT+O Add an Output to the selected Function.

CTRL+SHIFT+R Add an Input and Output pair (mapped together) to the selected Function (only for
                   the Input to Output Function).

F4                 Open the Properties tool pane.

F5                 Refresh the current view.

F11                Enable/disable the Full Screen view.

F12                Save the diagram as an image.

DEL                Remove the selected items.

CTRL+scroll Zoom in/Zoom out.
