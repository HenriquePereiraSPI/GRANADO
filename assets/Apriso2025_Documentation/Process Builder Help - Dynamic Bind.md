# Dynamic Bind

Dynamic Bind is used for binding an HTML element to a dynamically added Output. Name of the
Output can be controlled using an Input.

$Context can be used to access the value of the added Output.

<htmltag data-flx-dynamic-bind = " [Output name] "></htmltag>
