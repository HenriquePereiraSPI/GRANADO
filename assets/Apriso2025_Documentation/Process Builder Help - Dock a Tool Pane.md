# Dock a Tool Pane

Docking the tool panes is achieved by specifying a target destination or by docking the window into
a tabbed location.

To Move a Tool Pane to a New Location

       1. Select the tool pane you would like to move and start dragging it towards its destination.
           This will activate the navigation compass.

                The navigation compass enables you to dock a window at the required destination by
                placing the window over one of the points of the compass – both the ones in the middle
                of the screen and on the perimeter of the screen.
          2. Drag the tool pane onto a compass point. The screen display will indicate the potential
              target destination by outlining the area where the pane will be placed.

3. Release the mouse button over the compass point to confirm the destination and move the
   pane to its new location.
