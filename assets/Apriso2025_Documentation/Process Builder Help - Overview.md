# Overview

As the modeling component of DELMIA Apriso, Process Builder (PB) enables the creation of user
interfaces and business logic using the Screen Flow Management design methodology and entities
like Projects, Screens, and Data Flow Controls (DFC). It also facilitates interaction with
manufacturing operations, allowing easy structuring and modification of processes and operations.
With support for reusability, previously developed logic can be quickly adapted for new designs,
offering flexibility for frequent manufacturing operations changes. For more information on the
application, see What is Process Builder?.

A high-level overview of improvements and a list of Known Issues in Process Builder (and the
entire DELMIA Apriso product family) are described in the DELMIA Apriso Release Notes
document. The latest key features are also highlighted in the Welcome Screen (Help > Show
Welcome Screen).
