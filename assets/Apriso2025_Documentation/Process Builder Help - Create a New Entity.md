# Create a New Entity

To Launch the New Entity Wizard

Use one of the following options:

           Click (New...) on the toolbar.
           Select New... from the File main menu.
           Select New from the right-click menu in Entity Manager or the right-click menu in Project
           Entity Manager.
           Use the Ctrl+N keyboard shortcut.

When naming the entity, it is recommended to use the naming conventions discussed in the Best
Practices section.

  You can force naming conventions for an entity revision using the RevisionNamingConventionMask,
  DefaultRevisionName, or RevisionNamingConventionDescription keys located in the ProcessBuilder
  section of the Central Configuration file. For details, see Central Configuration
  Documentation .
