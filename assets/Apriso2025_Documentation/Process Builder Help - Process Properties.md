# Process Properties

The functionalities related to process authoring are deprecated and will not be further
developed.

To Launch the Process Properties Tool Pane

       1. In the Entity Explorer, select a process from the list.
       2. Choose one of the following options:

                      Use the F4 keyboard shortcut.
                      Choose View > Properties from the Main Menu.

You can also open the Entity Properties window by selecting a process or operation and using one
of the following options:

                      Choose Properties from the Right-Click Menu.
                      Use the Shift+F4 Keyboard Shortcut.
