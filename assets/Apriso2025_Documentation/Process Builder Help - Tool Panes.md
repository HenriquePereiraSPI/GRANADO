# Tool Panes

Some tool panes may not be available in a particular view, especially in the Entity Manager
  view.

The following tool panes provide you with additional functionalities while working with an entity:

           Entity Explorer – displays and allows you to navigate through the tree structure of an
           entity.
           Session Variables – displays all available session variables that you can use in DFCs.
           System Parameters – shows all system parameters along with the categories that they
           belong to.
           Validation Messages – displays the results of an entity's validation.
           Rich Text Description – displays a window with an extended description of a process,
           operation, DFC, step, screen, layout, and view.
           Categories – organizes processes, operations, DFCs, screens, layouts, and views in
           categories.
           Preview – gives the ability to preview the screen created in the HTML, Java Script, and
           CSS tabs within the DFC step.
           Change Orders – contains the list of change orders that affect a process or operation.
           Action Script Output – allows you to test and debug action scripts.
           Background Tasks – displays the progress of various background tasks.
           Recent and Favorite Items – displays a list of recently accessed entities and favorites.
           Toolbox – facilitates the adding and linking of entity elements.
           Unit Test Explorer – allows you to run unit tests for a DFC included in a project.
           Properties – displays and allows you to edit the properties of an edited entity.
           Advanced Find – allows you to find entities using keywords.

How To

   Display a tool pane
   Dock a tool pane
   Pin a docked tool pane
   Close a tool pane
