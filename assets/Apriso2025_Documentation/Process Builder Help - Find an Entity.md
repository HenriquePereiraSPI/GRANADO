# Find an Entity

To Display Advanced Find Tool Pane

Use one of the following options:

           Press Ctrl+F .
           Click (Advanced Find) on the toolbar.
           ChooseAdvanced Find from the Edit main menu.

To Use Advanced Find

Enter a text string into the search field and click Find or press Enter . You can choose to search

only the selected entity or all of them. Use  and                   to show or hide the advanced

find options. The tool works only for entities with saved changes.

  In the Advanced section, select the properties of entities or their components that you want to
  search. The tool searches for the text string in different fields depending on the selected check
  boxes.

   Components       Searched fields
   Process          Name, description, translation, code, and revision.
   Operation        Name, description, translation, code, and revision.
   DFC              Description, translation, code, revision, and properties.
   Step             Name and description.
   Function         Name, description, and other properties, for example SQL query.
   Function Input   Name and description.
   Function Output  Name and description.
   Screen           Name, description, and panel name.
   Layout           Name and description.
   View             Name and description.
   Action           Name.

Controls              Works only for actions defined for a screen.
Action Scripts
                    Name.
                    Name and function names.

Advanced Find Results

The results appear as a list in the Results window. Double-clicking the selected result redirects you
to the entity workspace, where the searched property is found, and highlights the component in
Entity Explorer. In addition, double-clicking the function, input, or output of the selected DFC
highlights them also in the function navigation diagram.
