Complex Types

Process Builder makes it possible to implement any functionality using simple data types, that is
Char, Integer, Decimal, Boolean, and Date Time. However, in case of large and complicated
projects, this may be insufficient and difficult to manage.

To address this problem and to allow better visibility and control over data that is passed around
within a DFC, a Complex data type has been introduced.

A Complex data type is a type that can group many simple variables and handle them as a single
object. Passing one Input or Output instead of numerous ones significantly simplifies Function and
DFC flows. It also improves the solution performance, reduces implementation time and simplifies
future modifications.
