# Invoke a DFC from a Step

1. Select the Step you want to add to the DFC to invoke.
       2. Go to the Advanced tab of the Properties tool pane of the step, and click the Add button.

           The pop-up window with a complete list of all DFCs available in the system appears.
       3. Select the DFC. It will be added to the list of invoke DFCs on the Advanced tab of the

           Properties tool pane for the step.
