# Entity Properties

The Entity Properties pane displays the properties of the following controls:

           Process
           Operation/Process Operation
           Operation Step
           DFC
           DFC Step
           Function
           Input
           Output
           Screen, Layout, and View (for details, see Managing Screen Flows)
           Action Script
           Project

To Open the Entity Properties Pane

While in the entity workspace, choose one of the following:

           Select Properties from the View menu.
           Click (Properties) on the toolbar.
           Use F4 keyboard shortcut.
