# Validation Messages

The Validation Messages tool pane displays the results of the validation of entities which has been
performed by pressing (Validate) on the toolbar or the Validate button in the navigation
workspace.

Double-clicking a validation message in the list highlights and navigates to the entity on the tree, on
the Function Navigation diagram, and in the Layout Editor.

It is possible to filter the results to display only errors, warnings, or messages by pressing
corresponding buttons. Validation Messages can be searched using the Find text box.

It is also possible to:

           Display a validation message in a separate window by right-clicking on the message and
           choosing Details from right-click menu
           Copy a validation message by right-clicking on the message and choosing Copy from the
           right-click menu
