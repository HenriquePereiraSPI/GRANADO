# Modify a Connector Line

To Draw a Connector Line

       1. Draw a connector line from one Step to another. When the source Step is a Decision Step,
           a dialog box appears prompting you to provide a Routing Value and a description, or to set
           the connector line as a default.

       2. The system updates the values and shows the value or description (if present) on the
           connector line.
            If the connector line is the default, it remains blank

  A Warning appears if there is no default exit from the Decision Step.

  The type of value entered must match the Routing Output type. Additionally, only one default
  routing can be set from a Step. A dialog box will be displayed to replace an existing default
  routing.

  When a connector line is deleted, the Routing Value is deleted as well.

To Break a Connector Line Between Two Nodes

  This feature can be used in complicated multi-Step flows to improve readability.
       1. Double-click on the connector line you want to break. The line will be highlighted in green,
           indicating that it has been selected. A rectangle will appear in the place where you clicked.
       2. Click on the rectangle to drag and drop the "joint" of the line to the desired location.

                   You can have multiple "joints" in one connector line.
