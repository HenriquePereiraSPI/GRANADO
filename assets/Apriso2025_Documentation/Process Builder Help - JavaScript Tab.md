# JavaScript Tab

The JavaScript tab allows for writing and including your JavaScript (functions, plugins, and
variables).

Depending on the scope of your functions, it may be defined in the following places:

     In the HTML tab
     In the JavaScript tab
     In an external file

Script    Runtime Behavior
Location

HTML tab The script will be generated per each instance of the DFC or HTML View on the
                   screen.

JavaScript Functions, plugins, and variables will be generated once per all instances of the

tab       same DFC (e.g., iterated sub-DFC) or HTML View on the screen.

External file Script links will be generated once per all instances of the DFCs or HTML Views
                   where it is included.

Refer to Scripting to see examples that present the cases mentioned above.

To Enable IntelliSense

To enable IntelliSense in the JavaScript tab, press CTRL+SPACE . A list of suggestions will appear.
