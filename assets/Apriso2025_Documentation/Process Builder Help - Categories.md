# Categories

The Categories pane is accessible from the View menu. Use categories to organize processes,
operations, DFCs, screens, views, and layouts in specific groups, such as production, warehouse,
or quality. A single entity can belong to many categories. Different entity revisions can also be a
part of different categories.

Categories have a hierarchical structure. They are available only in Entity Manager, not in PB
projects.

There are three main default folders:

           All – contains all entities available in Entity Manager.
           Unassigned – contains all entities that do not belong to any category.
           Categories – contains all your own category folders with entities that you assigned to
           them.

With no category selected, PB displays all entities. You can drag the category and move it
elsewhere on the tree.

Filtering                                                                                    (Clear

           Use the Find field to search for a specific category.
           Use the Quick Filter to find a specific entity. To clear all entered values, use
           Filter) or the Ctrl+E keyboard shortcut.

Categories Right-Click Menu

Add Category, Rename, Delete, and Properties options are available only for the categories
created inside the Categories folder.

                   Option    Shortcut Description

                   Add       Ins     Displays the Category Editor, where you can add a new
                                     category folder.
                   Category

                   Rename    F2      Changes the name of the selected category.

                   Expand All N/A    Expands the whole tree or the contents of the selected
                                     folder.

                   Collapse All N/A  Collapses the whole tree or the contents of the selected
                                     folder.

                   Delete    Del     Removes the selected category. Deleting a category does

                                     not remove its entities from the system.

                   Refresh   F5      Refreshes the content.

                   Properties Shift+F4 Displays the Category Editor with properties of the
                                                    selected category.
