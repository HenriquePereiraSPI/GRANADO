# General View Template_ Grid

General View Template: Grid

View Runtime

The Grid View template allows for using different configurations of a grid.

 Functional Overview

              View type: General
              DFC of a View type (the name of this DFC should be the same as the name of the View)
              Business Control used: Grid Business Control

 Configuration

          1. Copy and link the PortalGrid View Template on the Screen workspace:

          2. The following options are available:
                         Configuration of Business Controls (for details, see the Grid Business Control
                         settings)
                         Configuration of Buttons inside the Grid (for details, see the Toolbar Buttons Editor
                         in the Grid Business Control)
                         Configuration of Actions that are added to the View Grid as Buttons (for details,
                         see the Availability Mode and Group Tag properties on the View Action: General
                         Tab)
                         Configuration of click or double-click Actions on a row (for details, see the Display
                         Row Selector/Click Mode, Row Single Click Mode, and Row Double Click
                         Mode options on the Grid Business Control)
