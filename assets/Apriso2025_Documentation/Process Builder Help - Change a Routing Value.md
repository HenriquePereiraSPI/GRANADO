# Change a Routing Value

1. Right-click on the connector line containing the Routing Value that you want to modify, and
           choose Change Routing Value from the Right-Click Menu.

       2. A dialog box to edit the value and the description will be displayed.
       3. Enter the new Routing Value, and click OK to confirm the change.

The type of value entered must match the Routing Output type.

Only one default Output can be set from a Step. If the user tries to set two, an error message
will be displayed, and a choice between the two defaults will have to be made.

When a connector line is deleted, the Routing Value is deleted as well.
