# Copy User Controls

Copy a User Control

To Copy a User Control or Step Grid (Deprecated)

Drag-and-drop the User Control while holding the Ctrl key, or use the Copy and Paste option:

       1. Select the User Control you want to copy.
       2. Use one of the following options to copy the User Control:

                      Choose Edit > Copy from the Main Menu
                      Choose Copy from the Right-Click Menu
                      Use the Ctrl+C Keyboard Shortcut
       3. Use one of the following options to paste the User Control:
                      Choose Edit > Paste from the Main Menu
                      Choose Paste from the Right-Click Menu
                      Use the Ctrl+V Keyboard Shortcut
       4. The system places the User Controls from the clipboard into the design area:
                      If all User Controls were copied from the same grid, they are added at the end of
                      that grid
                      Otherwise, the User Controls are added at the bottom of the screen

The user can copy User Controls or Layout Controls:

           Within the Layout Editor
           From the Layout Editor to a Step in the Entity Explorer
