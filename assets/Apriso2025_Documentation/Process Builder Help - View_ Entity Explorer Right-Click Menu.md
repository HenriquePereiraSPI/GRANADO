# View_ Entity Explorer Right-Click Menu

View: Entity Explorer Right-Click Menu

View

                   Command           Description
                   Add Action        Adds an Action to the View.
                   Change Status     Changes the status of a View (for more
                                     information, see Entity Status Types).
                   Change Status     Recursively changes the status of a View.
                   Recursively

                                     Recursive change means changing the status
                                     of the entity and all its child entities.

                   Copy All Actions  Copies all Actions that are added to the View.
                   Expand All        Expands the list of all Actions for the selected
                                     View.
                   Collapse All      Collapses the list of Actions for the selected View.
                   Refresh           Refreshes the diagram.

Action

                   Command        Description

                   Disable        Disables an Action or enables the Action, depending in which
                   Action/Enable  state it is. If the Actions is in the disabled state, the Action icon
                   Action         is greyed-out (it cannot be used but you can change its
                                  properties).

                                                 This option can be available for Views that are in the active
                                                 status. For details on how to enable it, see Entity Status
                                                 Types.

                   Move Action Up Moves the Action down in the structure (available only if there
                                              is at least one Action below the selected one).

                   Move Action      This option can be available for Views that are in the active
                   Down             status. For details on how to enable it, see Entity Status
                                    Types.

                                  Moves the Action up in the structure (available only if there is
                                  at least one Action above the selected one).

                                  This option can be available for Views that are in the active
                                  status. For details on how to enable it, see Entity Status
                                  Types.

                   Rename   Changes an Action's name. This also renames any references
                            to this Action (e.g., in places where it is used as a Next
                   Cut      Action).
                   Copy     Copies and cuts the selected entity.
                   Paste    Copies the selected entity.
                   Delete   Pastes the copied entity.
                   Refresh  Deletes the selected entity.
                            Refreshes the diagram.
