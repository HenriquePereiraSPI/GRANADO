# Find_Replace dialog box

Find/Replace Dialog Box

The Find\Replace dialog box offers you the following functionalities:

           Find all of the occurrences of a word or string as defined in the Find what box
           Replace the occurrences of a word or string as defined in the Find what box with a new
           word or string as defined in the Replace with box

Field              Description

Match case If selected, the search action distinguishes uppercase and lowercase characters.

Match              If selected, only the entire occurrences of a word are included in the search. For

whole word example, if you search for "the," this option will ensure that words such as "theater"

                   or "theme" are not included in the search results.

Search up Select this to search backwards (towards the beginning of the document) from the
                  current cursor point.

Use                Indicates how to interpret the special characters entered in the Find what or

                   Replace with text boxes. The options include:

                   Regular expressions

                              When this option is chosen, you can use regular expressions in
                              your search criteria
                              For example, you could search segments that contain email
                              addresses in the text
                              Regular expressions are used to search text for all occurrences of a
                              particular sequence, or pattern, of characters
                   Wildcards

                                When this option is selected, you can use wildcards (special
                                characters such as asterisks "*"and question marks"?") in your
                                search criteria
                                Use a question mark ? wildcard to match a single character at the
                                current position
                                Use an asterisk * wildcard to match zero or more characters at the
                                current position

Search             If selected, this locates instances of the text that are concealed or collapsed.

hidden text

Search in If selected, this finds only the text within the selected text in the direction of the
selection search.

Find Next          Searches for the next occurrence of the characters that are specified in the Find
                   what box. To find the previous occurrence, hold down SHIFT and then click Find
                   Next.

Replace            Replaces the selected occurrence of the text in the Find what box, finds the next
                   occurrence, and then stops.

Replace All Replaces all of the occurrences of the search criteria in your document.

Mark All           Marks with a bookmark lines in the current active text editor file that contain the text
                   specified in the Find what box.

Close              Closes the Find/Replace dialog box.
