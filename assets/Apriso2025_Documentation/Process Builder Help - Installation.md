# Installation

DELMIA Apriso Process Builder is available in the DELMIA Apriso 2025 installation folder and is
automatically installed together with DELMIA Apriso on the server. The application can be installed
on a client machine using Microsoft® ClickOnce™ – a technology included in Microsoft® .NET™
Framework 4.8.

To Install Process Builder on a Client Machine

Click the Process Builder link in DELMIA Apriso Start page.

The system will invoke the ClickOnce™ engine to download and install the application. A menu
group called Dassault Systèmes is created in the Windows® Start Menu that contains a DELMIA
Apriso 2025 Process Builder menu item. The menu item name contains the name of the server
from which the application was installed and with which it is synchronized.

  A menu item will be created for each server instance from which you download the application.

Once the installation is complete, the system invokes the application, which will log you (using your
DELMIA Apriso login details) into the server from which it was installed.

If Process Builder has been installed previously and you try to install the same application from
another server, the system will display a pop-up window.

   You may choose to run the previously installed version, or install a new one from another server.

     If you intend to install the Process Builder version that had been installed before from the
     server, the system opens the previously installed application.

 To Run Process Builder After Installation

  Click on the Process Builder icon created in the Windows® Start Menu.

Only 100% Windows scaling setting is recommended. Other options might cause scaling
issues.
