# View Action_ Roles Tab

View Action: Roles Tab

The Roles added in the Roles tab will have access to the Action. If an employee does not have the
required Role, an error message will be displayed with information that the employee is not allowed
to access the Action.
Clicking (Add) invokes the Selecting Roles window with all the Roles listed. For more
information on defining Roles, refer to the Role Configuration topic in the Role and Skill
Configuration Help .
After selecting a Role, it appears in the list with its Name and Description.

  If no Roles are defined, the Action will be accessible to everyone.

  The Roles tab can be available for Views that are in the active status. For details on how to
  enable it, see "Modifying Properties..." section of the Entity Status Types topic.
