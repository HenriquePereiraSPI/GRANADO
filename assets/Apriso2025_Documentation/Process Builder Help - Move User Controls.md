# Move User Controls

Move a User Control

The user can change the order of controls and place them within the grids.

To Move a User Control Using Cut and Paste

       1. Select the User Control to be moved.
       2. Use one of the following options:

                      Choose Edit > Cut from the Main Menu
                      Choose Cut from the Right-Click Menu
                      Use the Ctrl+X Keyboard Shortcut
       3. Use one of the following options to paste the User Control:
                      Choose Edit > Paste from the Main Menu
                      Choose Paste from the Right-Click Menu
                      Use the Ctrl+V Keyboard Shortcut

To Move a User Control or Step Grid (Deprecated) Using Drag-
and-Drop

       1. Select the User Control to be moved.
       2. Drag-and-drop the User Control within the design area into the chosen location.

           The user can move the User Control or Step grid:

                      Within the Layout Editor
                      From the Layout Editor to a Step in Entity Explorer
