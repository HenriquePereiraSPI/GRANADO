# Input Source

Inputs may be obtained from one of several sources. The list of valid sources is as follows:

           User – an Input from the screen
           Previous Function – points to another Function's Output from the same Step
           Session Variable – the name of a Session Variable or System Variable defined in the same
           DFC
           Constant – a constant value
           Machine – a Machine Integrator Input (local or remote)
           System Parameter – a global (system-wide) constant that is available in all of the business
           logic on the server, and which can be used to change the behavior of DFCs and
           Processes that have already been released
