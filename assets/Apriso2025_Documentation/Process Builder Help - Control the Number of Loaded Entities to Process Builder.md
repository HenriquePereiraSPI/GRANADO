# Control the Number of Loaded Entities to Process Builder

Control the Number of Entities Loaded to Process

Builder

       1. Select Tools > Options from the Main Menu.
       2. Choose the Advanced tab.
       3. Use one of the following options from the Entities loading mode section:

                      Select Do not load cancelled entities if there are more than 1500 entities of
                      one type (this is the default option) in order to limit the number of loaded entities
                      to Process Builder
                      It is also possible to change the number of entities loaded with use of the
                      "MaxNumberOfEntitiesToLoad" key located in the "ProcessBuilder" section of the
                      Central Configuration file (for details see Central Configuration Documentation )
                      Select Always load entities in order to load all entities regardless of the total
                      number of entities
