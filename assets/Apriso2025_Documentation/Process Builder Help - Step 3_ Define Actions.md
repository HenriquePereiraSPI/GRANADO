# Step 3_ Define Actions

Step 3: Define Actions

To define an Action, choose one of the options described below.

Add a New Action to an Existing Panel

In this option, you can add a new Action to an existing Panel on a Screen with a linked View using
the right-click menu:

  The Action is added under that particular Panel:

 Use an Action Inherited from a Linked View

 Methods

  The different methods for using this option to define an Action are described below.

  Use an Original Action from the View

In this case, you cannot change any properties of this Action (the Properties pane options are
greyed-out).

Override the Original Action

After overriding the Action you can then change its properties, which then become the properties of
this Action on the Screen.
Additionally, after overriding the Action, the icon changes:

Disable the Action

To disable the Action, first you need to override it.

     After disabling the Action, its icon is greyed-out. It thus cannot be used, but you can change its
     properties.
