# Manage Operations

Add a New Process Operation
           1. Choose Add New Operation from the right-click menu.
           2. Fill in all fields in the New Operation pop-up window:

              3. Click OK.
      The system adds the new operation to the diagram and Entity Explorer tree.
      Select an Operation

     To Select a Single Operation

      Use one of the following options:
                 Click the operation or draw a box around it in the operation navigation workspace.
                 Select the Operation in the Entity Explorer tree.

     To Select a Group of Operations

      Use one of the following options:
                 Draw a box around the operations that you want to select.
                 Click the operation that you want to select while holding down the Ctrl key.

      The selected operations will become highlighted and move together as a group.
      Move an Operation

     To Move a Single Operation

              1. Select the operation that you want to move.
              2. Drag and move the operation in the operation navigation workspace.

     To Move a Group of Operations

              1. Select the group of operations that you want to move.
              2. Drag and move the group in the operation navigation workspace.
