# System Parameters

The System Parameters tool pane shows all of the System Parameters along with the System
Parameter Categories they belong to.

                   Field          Description
                   Filter
                                  Enter the name of the System Parameter or
                   Show used      System Parameter Category that you want
                   onlycheck box  to find.

                                  If selected, only System Parameters that are
                                  used in the opened entity are displayed.

 Drag-and-Drop Actions Available on System Parameters

              From the pane to the Function's Input – the system changes the Input source
              From the pane to the Function, between Inputs – the system creates a new Input with the
              source set to the existing System Parameter
              It is also possible to drag-and-drop multiple System Parameters
              When dragging-and-dropping a System Parameter, an Input is created based on the
              System Parameter data type
              Validation returns an error when an Input data type is not compatible with a System
              Parameter data type

  It is possible to see where a System Parameter is used. After selecting a System Parameter in the
  System Parameter tool pane, Entity Explorer highlights the Function Inputs which use this variable
  (parent entities are highlighted if children nodes are collapsed in order to facilitate navigating to
  specific Function Inputs).

  For information about the System Parameter Input source, see System Parameter.

  It is possible to refresh the view in System Parameters by right-clicking on the system parameter in
  the tool pane and choosing Refresh from the right-click menu.
