# Open an Existing Entity

Open/Save/Close an Entity

To Open an Entity

       1. Use one of the following methods to launch the Open entity pop-up window:
                      Select the Open... option from the Main Menu
                      Click (Open) on the Toolbar
                      Use the Ctrl+O keyboard shortcut

       2. Start typing the name of the Entity or a part of it. Note that periods and underscores can be
           omitted in the search.

       3. Select the Entity and click Open or use double-click.

     When you open an entity that is checked out by another user, the system will only allow you to
     view it in read-only mode.

     The Entity Manager (Managers > Entity Manager) can also be used to open an existing
     document and offers more functions than the Open dialog box (e.g., filtering, sorting).

 To Save an Entity

  Use one of the following options:
              Select the Save option from the Main Menu
              Click (Save) on the Toolbar
              Use the Ctrl+S keyboard shortcut

An action comment may have to be entered when saving a process or operation. This feature is
related to electronic signatures and can be enabled using the ShowNoteOnSave key located in the
ProcessBuilder section of the Central Configuration file (for details, see Central Configuration
Documentation ). Action comments should describe the changes made to the entity since the
last save. The history of action comments is visible in the History tab of entity properties.

To Close an Entity

Use one of the following methods:

           Click (Close) on the Toolbar
           Choose Window > Close
           Click the X in the upper-right corner of the entity tab
           Use the Ctrl+W or Ctrl+F4 shortcuts

If changes were made to the entity, a dialog box with a saving confirmation will be displayed.
