# HTML View_ HTML Tab

HTML View: HTML Tab

HTML tab allows creating user interface using HTML tags, special attributes, Inputs, Outputs,
Screen Literals, Font Icons and Conditionals.

Adding Actions

Actions defined for the View can only be added using Action, Button and Tab attributes in the HTML
tab.

  If an Action is disabled in the Entity Explorer, it should be removed from the HTML tab.

Adding Inputs

To add an Input:
       1. Type a valid name of the Input inside curly brackets, e.g., {input}
           Input of the name "input" should be added to the HTML View Inputs grid.

       2. Save changes to the HTML View.

or

       1. Create an Input in the HTML View Inputs grid.
       2. Drag the Input from the grid to the HTML tab.
