# Tab Text

This attribute is only available in HTML View.

<htmltag data-flx-tab-text = "[Input name]" ></htmltag>

This attribute can be used to display extra text in the tab, like the number of items available on the
list that opens after clicking the tab.
Use an HTML View Input to provide text for the tabs. Use list of char Input to provide values for
multiple tabs.
