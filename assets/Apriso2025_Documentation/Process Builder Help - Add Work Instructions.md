# Add Work Instructions

You can link a work instruction to a process operation, operation, or operation step.

       1. Go to the Work Instructions tab of the properties tool pane for a process operation,
           operation, or operation step.

       2. Click (Link Work Instructions).
       3. Select one or more work instructions (hold down the Ctrl key) and click Finish. Work

           instructions in status Cancelled are not displayed in the list.
       4. Select the Link as dynamic revision check box if you want work instructions to appear

           using the dynamic revision algorithm.

  The work instructions are added to the list on the Work Instructions tab in the properties tool pane
  for a process operation, operation, or operation step
