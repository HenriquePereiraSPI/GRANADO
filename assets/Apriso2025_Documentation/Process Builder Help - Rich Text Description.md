# Rich Text Description

You can define a rich text description of a category, process, operation, DFC, step, screen, view,
and layout. Rich text description appears in a separate tool pane, which includes a window for an
extended description.

You can define templates for the description of a category, process, operation, DFC, step, screen,
view, and layout. Templates may be changed according to specific needs. Changes in templates
affect only the entity descriptions that have not been altered. Altering an entity description causes
the creation of a specific description instance to no longer be linked to the template.

  The content of the rich text description can be defined in the HTML editor. Importing an HTML file
  that can be used as a rich text description is also possible.
  You can export the description prepared in the HTML editor and save it as an HTML file.
