# Layout Editor Right-Click Menu

This can be accessed by using the right-click menu keyboard key or by right-clicking anywhere in
the Layout Editor workspace.

                   Command     Description

                   Rename      Allows the user to change the name of the Control.

                   Open Sub-DFC Opens the Sub-DFC Function in a new tab (available only for
                                          Functions of the Sub-DFCs type).

                   Cut         Executes the functionality to cut (the first part of the Move action)

                               the selected Control. The user can select more than one Control to

                               be cut.

                   Copy        Executes the functionality to copy (the first part of the Copy with
                               Paste action) the selected control. The user can select more than
                               one control to be copied.

                   Paste       Executes the functionality to paste the control from the clipboard.
                               This is the second part of both the Cut and Copy actions.

                   Delete      Deletes the selected Control.

                   Duplicate   Available only when a Control is selected. The user can select
                   Selection   more than one Control for duplication.

                   Save as Image Saves the Layout Editor diagram as an image.

                   Properties  Opens the Properties tool pane for a Step or for the selected
                               Control.

                   Convert to  Converts the Layout created in Layout Editor to HTML Layout. After

                   HTML Layout conversion Layout Editor will be not available.
