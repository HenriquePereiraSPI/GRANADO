# General View Template_ Visualization

General View Template: Visualization

View Runtime

The Visualization View template allows for the configuration and display of a chart or gauge.

 Functional Overview

              View type: General
              DFC of View type (the name of this DFC should be the same as View)
              Business Control used: Visualization Business Control

 Configuration

          1. Copy and link the PortalVisualization View Template.

2. Use the settings available in the Visualization Business Control.
