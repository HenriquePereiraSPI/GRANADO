# Apriso Message Function

The Apriso Message Function enables publishing messages within DELMIA Apriso to interact with
other DELMIA Apriso elements.

The function outputs a message in JSON format.

To add the Apriso Message Function

       1. Drag the Apriso Message Function from the Toolbox to the Function Navigation diagram,
           or double-click the Show Message icon in the Toolbox.

       2. Check Message in JSON Format to change the function's input format from a List to
           JSON.

Apriso Message Function Inputs

Message in List Format

Input              Description
Topic              The topic of the message to be published.

FieldList          List of keys.
ValueList          List of key values.

Message in JSON Format

Input              Description
Topic              The topic of the message to be published.

   Message         Message in JSON format.
