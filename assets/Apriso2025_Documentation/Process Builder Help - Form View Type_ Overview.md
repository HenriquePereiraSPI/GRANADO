# Form View Type_ Overview

Form View Type: Overview

View Runtime

The Form View type allows for building a user interface with various controls such as text boxes,
drop-down lists, combo boxes, time pickers, etc.

 Functional Overview

              View type: Form
              DFC: GenericPortalForm
              No Business Control is required

 Configuration

          1. Copy and link the PortalForm View Template in the Screen workspace:

2. The following configuration options are available for the linked View:

              Adding Buttons with Actions (for details, see View Workspace: Forms Tab)
              Adding groups (for details, see View Workspace: Forms Tab)
              Adding controls within groups (for details, see View Workspace: Forms Tab)
