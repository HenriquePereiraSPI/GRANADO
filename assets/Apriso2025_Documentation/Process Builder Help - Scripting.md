# Scripting

JavaScript script in the HTML tab will be included for each instance of this DFC.

  JavaScript script in the JavaScript tab will be included only once per all instances of this DFC.

  JavaScript files linked in the JavaScript tab will be included only once for all DFCs. Script links
  will be merged based on src attribute (case insensitive).

  If you have many different DFCs using the same JavaScript functions, save them into one
  external file and link it in the JavaScript tab.
