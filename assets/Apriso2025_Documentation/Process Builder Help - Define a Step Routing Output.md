# Define a Step Routing Output

Setting the routing output on a step defines the navigation flow of the step. Test Run uses the
defined output value to run the DFC according to the designed path of step navigation.

       1. Select the step in which the routing output is to be defined.
       2. Right-click the step and choose Set Routing Output from the right-click menu. The

           Function Outputs window appears.
       3. Select an output from any of the available functions. Click OK to return to the navigation

           diagram.

              The types of the allowed outputs: Char, Integer, Boolean, and Decimal.

Steps with a set routing output become decision steps with a indicator displayed on the step
information bar.
