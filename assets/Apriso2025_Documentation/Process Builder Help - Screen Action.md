# Screen Action

Screen Actions behave the same way as View Actions.
Once a View is linked to Screen Panel, its Actions are shown on the Screen Entity Explorer pane.

Additionally those View Actions can be:

           Overridden on the Screen (the Action then becomes a local Action for the specific Screen
           with modified properties)
