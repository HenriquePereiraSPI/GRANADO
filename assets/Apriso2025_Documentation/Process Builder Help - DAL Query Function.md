# DAL Query Function

To Add the DAL Query Function

       1. Drag the DAL Query Function from the Toolbox to the Function Navigation diagram, or
           double-click the DAL Query icon in the Toolbox.

       2. The system displays a complete list of DAL methods which the user is able to choose
           from. When the user enters a part of the DAL class or method, the browser suggests a list
           of matching entries from the database.

          3. After the selection of the DAL method, the system adds Inputs to the Function based on
              the Inputs of the method.

          4. The user can choose which method's Outputs will be bound to the Function Outputs. The
              user can select and deselect all Outputs for binding in the Function's Properties Pane.

Field               Description
Assembly Name       The name of the assembly containing the Data Access Layer
                    method.
Class Name          The name of the Class containing the DAL method.
Method Name         The name of the method to invoke.
Link button         Links other DAL method.
