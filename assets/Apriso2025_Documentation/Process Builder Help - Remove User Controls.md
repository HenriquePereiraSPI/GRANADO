# Remove User Controls

Remove a User Control

To Remove a User Control

       1. Select the User Control to be removed.
       2. Use one of the following options:

                      Choose Edit > Delete from the Main Menu
                      Choose Edit > Cut from the Main Menu
                      Choose Delete or Cut from the Right-Click Menu
                      Use the Del Keyboard Shortcut

The System removes grids together with their contents.

Removing User Controls means that Business Control Functions are removed as well and related
Inputs and Outputs will be affected. Inputs and Outputs that have been automatically created and
not manually modified will be removed. In other cases, the Input source is changed to constant or
the Output routing will be removed.

Removing a Sub-DFC from Layout Editor means that it is removed from the Step.
