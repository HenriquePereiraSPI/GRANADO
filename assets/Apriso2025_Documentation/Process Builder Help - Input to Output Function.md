# Input to Output Function

This type of Function is used to pass values from Inputs to Outputs, depending on the mapping
between them.

To Add the Input to Output Function

       1. Drag the Input to Output Function from the Toolbox to the Function Navigation diagram, or
           double-click the Input to Output icon in the Toolbox.

          2. Add pairs of Inputs and Outputs to the Function (see Create Input or Output). When Inputs
              and Outputs are added separately, they should be mapped together in the Function's
              properties.

 To Map Inputs with Outputs

          1. Select an Output on the grid in the Function properties.
          2. Choose an Input from the drop-down list. The Input's value will be passed in runtime to the

              Output.
                Only Inputs and Outputs of the same data type can be mapped together. Changing the
                type of an Input or Output will also change the type of the mapped items.

                   An Input and Output that are mapped together are displayed beside each other. When
                   there are multiple Outputs mapped to one Input, only the first Output is displayed in the
                   diagram beside the Input.
