# Button Attribute

Button

  This attribute is only available in HTML View.

   <htmltag data-flx-button = "[action names]" ></htmltag>

Shows the specified Actions. Allowed values:

           * or no value - show all Actions
           "actionname" - show only the specific Action
           "actonname1,actionname2" - show listed Actions
           "!actionname" - shows all actions except specified Action
           "!actonname1,!actionname2" - shows all except listed Actions

If an Action is disabled in the Entity Explorer, it must be removed from the HTML tab.
