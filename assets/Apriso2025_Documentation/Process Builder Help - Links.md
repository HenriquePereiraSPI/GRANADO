# Links

Links Workspace Overview

The Links tab is a workspace available for processes, process operations, operations, and
operation steps.

Available Links                                         Process Process Operation Step
                                                                       Operation
Link \ Available Target

   Characteristic
   Component
   Document
   Product
   Resource (resource types: Generic, Building,
Container, Equipment, Land, Supplier, Collection)
   Specification (you can link only one specification)
   Work Center (you can link only one work center)

Links Tab Right-Click Menu

                   Command Description

                   Link  Adds a component from the list.

                   Preview Previews the linked document in the appropriate viewer
                                 (for example, Adobe® Reader® ).

                   Cut   Cuts and copies the selected link.

                   Copy Copies the selected link.

                   Paste Pastes the copied link.

                   Delete Removes a link.

                   Details Displays the properties of a resource, component, or
                                 characteristic.

To Add a Link

       1. Display the link entity wizard using one of the following options:

                      Select the entity or step on the Entity Explorer and drag the link from the
                      Toolbox to the links workspace.
                      Drag a link from the Toolbox to the entity in the Entity Explorer.
                      Use the right-click menu and choose one of the links from the Link option.

       2. Depending on your choice, you can link:

               Characteristic
               Component
               Document
               Product
               Resource
               Specification
               Work center

       3. The link is added to the workspace and to the Recent and Favorite Items tool pane.

To Remove a Link

Select the link and use one of the following options:

           The DEL key.
           The Delete option from the right-click menu.

To undo a link removal, select Undo from the Edit menu, use the Ctrl+Z keyboard shortcut, or
click (Undo) in the toolbar.
