# HTML View_ CSS Tab

HTML View: CSS Tab

The CSS tab allows for writing and including your CSS styles.

Depending on the scope of your styles it may be defined in the following places:

           In the HTML tab
           In the CSS tab
           In an external file (for a specific Theme)

Styles             Runtime Behavior
Location
HTML tab           Styles will be generated per each instance of the DFC or HTML View on the screen.
CSS tab            Styles will be generated once per all instances of the same DFC (e.g., an iterated
                   sub-DFC) or HTML View on the screen.
External           Style links will be generated once per all instances of the DFCs or HTML Views
file               where it is included (files need to be included in a specific Theme folder/folders).

Refer to Styling to see examples that present the cases mentioned above. Refer to How to Use
Font Icons for examples of using icon classes.
