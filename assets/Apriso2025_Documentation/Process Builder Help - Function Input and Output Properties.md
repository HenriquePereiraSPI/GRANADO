# Function Input and Output Properties

The system allows the user to edit the Input and Output properties in the Properties pane.

To Launch the Properties Pane

       1. Select the Input or Output in the diagram or in the Entity Explorer.
       2. Choose one of the following options:

                      Choose Properties from the Function Navigation Right-Click Menu
                      Use the F4 Keyboard Shortcut
                      Choose View > Properties from the Main Menu
