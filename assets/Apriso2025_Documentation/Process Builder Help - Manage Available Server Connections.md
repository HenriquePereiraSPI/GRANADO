# Manage Available Server Connections

Servers are managed in the Apriso Servers box, which is displayed upon clicking the button
on the main login box.

System administrators can prepopulate the list of available servers using
the ServersSettings.xml file available under <drive>\Program Files\Dassault Systemes\DELMIA
Apriso 2025\WebSite\Downloads\PB2 on the DELMIA Apriso server. All the necessary instructions
are included in the file.

 To Add a Server to the Available List

          1. Click the Add... button to invoke the Apriso Servers box for entering the required server
              information.

       2. Enter an alias for the server and a correct address string.

              When using 3DPassport authentication, a full domain name for the server is required.

              If the server supports SSL encryption, select the Use SSL check box.
       3. Click OK to finish adding the server, which will now appear on the list of available servers.

To Change the Properties of a Server on the Available List

       1. Select the server you would like to change the properties for, and click the Change...
           button to launch the box for changing the required server information.

2. Make the desired changes and click OK to confirm.

To Remove a Server from the Available List

Select the server you would like to remove and click the Remove button. A dialog box will appear
to confirm the deletion of the server.
