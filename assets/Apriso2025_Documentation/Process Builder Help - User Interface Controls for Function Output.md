# User Interface Controls for Function Output

There are different types of Output Controls

User               Description
Output
Control            Displays the value as regular text.
Text               Displays the HTML output without replacing special characters. Plain HTML Output can
HTML               be switched using the "RenderPlainHtmlOutput" key located in the
                   "FunctionInterpreter" section of the Central Configuration file (for details, see Central
Link               Configuration Documentation ).
                   Displays the value as a link or embedded content. See the Display Mode option.

Properties

On the right side of the Layout Editor workspace, the system displays the properties of Inputs,
Outputs, and Grids. When the user selects an element in the central part of the screen, the system
shows an appropriate editor in the properties pane. For more information about the properties of
Output Controls, see Output Destination: User.
