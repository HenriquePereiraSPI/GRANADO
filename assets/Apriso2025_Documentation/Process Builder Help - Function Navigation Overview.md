# Function Navigation Overview

The Function Navigation Editor can be opened by drilling down below the Step level using the
double-click or Enter button:

           From the Step Flow diagram
           From the Entity Explorer – on Step, Functions, Inputs, or Outputs

It is possible to move between Functions or between a Function's Inputs or Outputs in Function
Navigation using the arrows on the keyboard.

The Function Navigation diagram provides a graphic representation of the content of a Step.

The Functions are presented as blocks with Inputs and Outputs on both sides
Each Function type is marked with a different icon, and various types of Input sources and
Output routing dispositions are indicated with different colors (see table below)

Background Color   Input Source         Output Routing Disposition

                   Constant             -

                   User                 User

                   Machine              Machine

                   User External Input  External

                   System Parameter     -

                   Session Variable     Session

The list of data types and their processing modes are marked with additional icons
Arrows between Inputs and Outputs determine the flow of the data
