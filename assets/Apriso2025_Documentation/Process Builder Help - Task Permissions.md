# Task Permissions

Manage Task Permissions

You can add task permissions to process operations, operations, or their steps. There are four
types of task permissions:

               Skill
               Role
               Employee
               Employee class

                Employee class is defined by the resource class of the person type.

 How To

      Add a Task Permission to an Operation

      Use one of the following options:

                 When creating a new operation, you can add task permissions in one of the steps.
                 Right-click the operation in Entity Manager right-click menu, choose the Properties
                 option, and go to the Task Permissions tab.
                 Drag a task permission from the toolbox to the operation in the Entity Explorer.
                 On the operation properties pane, go to the Advanced tab, and click Task
                 Permissions.

      Add a Task Permission to a Process Operation

      Use one of the following options:

                 Drag a task permission from the toolbox to the process operation in the Entity
                 Explorer.
                 On the process operation properties pane, go to the Advanced tab, and click Task
                 Permissions.

      Add a Task Permission to a Step

      Use one of the following options:

                 Drag a task permission from the toolbox to the step in the Entity Explorer.
                 On the step properties pane, go to the Advanced tab, and click Task Permissions.

      Copy a Task Permission
              1. Use one of the following options:
                            Hold the Ctrl key and drag the task permission from the source location to the
                            target one in Entity Explorer.
                            Select the link in Entity Explorer and:
                                        Use the Ctrl+C and Ctrl+V keyboard shortcuts.
                                        Choose Copy and Paste from the Edit menu.
                                        Click (Copy) and (Paste) in the toolbar.
                                        Choose Copy and Paste from the Entity Explorer right-click menu.
              2. A copy of the task permission appears in the target location.

Move a Task Permission

1. Use one of the following options:

                   Drag the task permission from the source location in Entity Explorer and drop
                   it in the target location.
                   Select the task permission in Entity Explorer and:

                          Use the Ctrl+X and Ctrl+V keyboard shortcuts.
                          Choose Cut and Paste from the Edit menu.

                          Click (Cut) and (Paste) in the toolbar.
                          Choose Cut and Paste from the Entity Explorer right-click menu.
2. The link disappears from the source location and reappears in the target one.

Remove a Task Permission

       1. Use on of the following options:

                      Select the task permission in the Entity Explorer and use one of the following
                      options:

                                 Press the Del key.
                                 The Delete option from the right-click menu.
                      Select the entity or step in the Entity Explorer, and on the properties pane, go
                      to the Advanced tab, click Task Permissions, select the task permission, and
                      use Remove.

To undo a task permission removal, select Undo from the Edit menu, use the Ctrl+Z
keyboard shortcut, or click (Undo) in the toolbar.
