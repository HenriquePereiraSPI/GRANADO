# Check an Entity In_Out

Check an Entity In/Out

To Check Out (Lock) an Entity

       1. Launch Entity Manager or Project Entity Manager.
       2. Select the entity to check out from the list.
       3. Use one of the following options:

                      Click (Check Out) on the toolbar.
                      Choose Check Out from the Entity main menu.
                      Select Check Out from the Entity Manager right-click menu or Project Entity
                      Manager right-click menu.

If you lock an entity that is open in a tab, the lock is automatically released when you close the tab.

After checking out the entity, your login name appears in the Locked By column in Entity Manager
or Project Entity Manager.

If the entity is already locked by another user, you can:

           Open the entity in the read-only mode. In the case of processes, you can still print or save
           the operation navigation diagram. In the case of DFCs, you can still save the step
           navigation diagram.
           Copy your changes manually to another entity.
           Check out the locked entity. The other user will not be able to save their work and the
           entity will become locked by you.

To Check In (Unlock) an Entity

       1. Launch Entity Manager or Project Entity Manager.
       2. Select the entity to check in from the list.
       3. Use one of the following options:

                      Click (Check In) on the toolbar.
                      Choose Check In from the Entity main menu.
                      Select Check In from the Entity Manager right-click menu or Project Entity
                      Manager right-click menu.

The entity will be unlocked and your login name disappears from the Locked By column in Entity
Manager or Project Entity Manager.
