# Step 1_ Link a Layout to a Screen

Step 1: Link a Layout to a Screen

Create screens based on a layout as described in Getting Started. The layout will be then linked
automatically. Use the following steps if a screen is not linked to a layout (the screen was created
using a different method) or to link a new layout to an existing screen.

To Link a Layout to a Screen

       1. Use one of these methods:
                      Click (Link) on the Screen: General Tab

                         Right-click the screen in Entity Explorer and select the Link Layout option

                   Click  on the screen workspace (this option is available if a screen is not

                   currently linked to a layout)

2. Choose the layout to link from the Layouts pop-up window.
      You can choose a predefined one from templates.

              After the layout is linked it is not added as a new entity itself in Entity Explorer because
              only panels are added under the screen. However, the name of the linked layout is added
              to the name of the screen:

For more information on layouts, see Layout Overview.
