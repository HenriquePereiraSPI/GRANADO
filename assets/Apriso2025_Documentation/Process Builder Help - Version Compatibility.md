# Version Compatibility

Conflicts can occur when the version of Process Builder components installed on the client
machines differs from that of the corresponding components on the server.

The system performs a version compatibility test each time you log in to Process Builder, and
displays a dialog box to inform you of incompatibility when detected. The version information of
particular DELMIA Apriso Components can be compared with use of the Component dialog box
accessible through Help > About > Components.

  The Client and Server Versions should be exactly the same, as should the File Versions for
  server and client.

Though the system may allow you to work with two incompatible versions, it is strongly
recommended that you identify and update the incompatible machine to the latest version. See the
Installation page for more details.
