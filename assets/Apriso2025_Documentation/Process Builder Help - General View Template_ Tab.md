# General View Template_ Tab

General View Template: Tab

View Runtime

 Functional Overview

              View type: General
              DFC of a View type: GenericPortalTab
              No Business Control is required

 Configuration

          1. Either simple Tabs or notification Tabs can be configured:
                         For simple Tabs, copy and link the PortalTabView Template
                         For notification Tabs, which are able to display notification numbers (like the Alert
                         List Tab in the image above), copy and link the PortalTabNotificationView
                         Template
                                    The NotifyCountList Input can be configured in the DFC linked to this
                                    View, and the numbers passed to this Input are displayed on subsequent
                                    Tabs as notification numbers:

          2. The following options are available:
                         Configuration of Actions of the Tab type
                                    The number of Tabs to appear is the number of Tab Actions in the View
                                    (for details, see View Action: General Tab)
                         The default (active) Tab is the one you set in the Default Action menu in the
                         Screen Panel properties (for details, see Panel: General Tab)
