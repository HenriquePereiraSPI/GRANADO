# Recent and Favorite Items

This tool pane lets you:

           Create a list of favorite operations or operation steps.
           Access recently added links.

Actions

           To display the tool pane, click the View menu and select Recent and Favorite Items.
           To add one or more operations or operation steps to the list, drag them to the tool pane.
           When you add a link to the entity workspace, the link automatically appears in the tool
           pane.

Recent and Favorite Items Right-Click Menu

Command            Description
Clear All          Clears the list.
Delete             Removes the selected item.
