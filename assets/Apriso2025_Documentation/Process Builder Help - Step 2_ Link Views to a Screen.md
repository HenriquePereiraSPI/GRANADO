# Step 2_ Link Views to a Screen

Step 2: Link Views to a Screen

To Link a View to a Screen

       1. Link a View by using one of the following options:
                      Click (Copy And Link View) on the Screen workspace (recommended).

                   When using the Copy And Link View functionality, the copy of the View and
                   its linked DFC is created automatically.

                         Click (Copy & Link) on the General Tab of the Panel Properties pane.

          2. Choose the View to link from the Views pop-up window.

                   Using predefined Templates is recommended (for details, see View Templates).

              The basic configuration of the screen is now done. After the View is linked it is not added
              as a new entity itself in Entity Explorer, because only Panels are added under the Screen.
