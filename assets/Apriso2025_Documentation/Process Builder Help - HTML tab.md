# HTML tab

The HTML Layout provides the user with the ability to:

           Use the already defined CSS Classes in the client mode. For details, see the CSS
           Framework Documentation .
           Design the layout of a screen using HTML, CSS and JavaScript, as it contains:

                      Standard Apriso controls: Business Controls, User Inputs and Outputs, Sub-DFCs,
                      Step Grids (Controls Tags Overview)
                      Special attributes
                      Any other HTML element
           Read and write values from and to Process Builder Inputs and Outputs (see Screen
           Interface Input and Screen Interface Output)

  HTML is compatible with the XHTML 1.1 standard.

To Enable CSS Class Suggestions

When inside an HTML class attribute, you can press CTRL+SPACE to enable CSS class suggestions
(IntelliSense). For this functionality to work properly, however, the cursor must be inside the
attribute's quotation marks.
