# Input Source_ Previous Function

Input Source: Previous Function

The general properties available for all inputs are described in the Input Properties topic.

  Previous Function Output is a combo box with a list of all of the defined Function Outputs that
  match the Input data type (this list contains only Outputs with data types convertible to the Input
  data type). The Function Input as Output from Previous Function works in the context of a single
  Step.

     If you intend to share Inputs and Outputs between Steps, you should be using a Session
     Variable and not a Output from Previous Function.
