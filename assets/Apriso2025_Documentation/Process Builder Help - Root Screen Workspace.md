# Root Screen Workspace

The screen workspace presents the actual phase of a screen configuration. After creating the
screen, the workspace suggests defining the layout.

When the layout with its panels is linked to a screen, you can easily observe its graphic design in
the workspace. Next, you can:

           Link views to panel.
           Open the view that is already linked.
           Open a DFC that is linked with the view.
           See the properties of actions linked to views.
