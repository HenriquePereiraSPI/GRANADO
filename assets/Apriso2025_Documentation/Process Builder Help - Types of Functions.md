# Types of Functions

Types of Function Description

3DEXPERIENCE Enables publishing messages from DELMIA Apriso to the

Message            3DEXPERIENCE Platform using an ActiveMQ Message Broker. For more

                   information, see 3DEXPERIENCE Message Function.

Apriso Message Enables publishing messages within DELMIA Apriso to interact with other
                          DELMIA Apriso elements. For more information, see Apriso Message
                          Function.

Business           Invokes a Business Component. A complete list of Business Components are
Component          displayed in the Business Component Repository. For more information, see
                   Business Component Function.

Business Control Displays a Business Control, which is similar to a Business Component but
                          with a User Interface. Additional configuration options are displayed
                          depending on the chosen Business Control. For detailed information about
                          Business Controls, please see Business Controls.

DAL Query          Invokes a DAL method to retrieve rows from the database. A complete list of
                   DAL methods is displayed to the user. This query is portable between MS
                   SQL and Oracle databases. For more information, see DAL Query Function.

Determination      Used to generate Outputs based on a combination of values from Inputs. For
                   more information, see Determination Function.

Input to Output Used to pass values from Inputs to Outputs, depending on the mapping
                          between them. For more information, see Input to Output Function.

Local              This is an expanded version of Determination. The Inputs, Outputs, and their
Determination      values are defined for the whole system, not for a particular Function. It is
                   possible to set date ranges for when particular values of Local Determination
                   are valid. For more information, see Local Determination Function.

MDX Query          Executes a user-defined MDX query against the configured database. For
                   more information, see MDX Query Function.

Message Broker Enables publishing messages from DELMIA Apriso to an external message
                          broker (Kafka, MQTT, or AMQP). For more information, see Message Broker
                          Function.

Notification       Enables configuring push notifications that can be sent from the DELMIA
                   Apriso server to the DELMIA Apriso mobile app. For more information, see
                   Notification Function.

MIScript           Executes a script which operates on Machine Integrator data. For more
                   information, see MIScript Function.

Show Message       The Show Message Function is used to define a set of messages that will be
                   shown on the screen. It is typically used for error messages. For more
                   information, see Show Message Function.

SQL Query          Executes a user-defined SQL query against the configured database. The
                   Function Inputs can be used as parameters of the query, while the Outputs
                   are populated with data from related query columns. For more information,
                   see SQL Query Function.

Stored Procedure Executes a Stored Procedure against the configured database. For more
                          information, see Stored Procedure Function.

Sub-DFC            Executes a DFC. The DFC to be executed can be selected by name (and
                   optionally revision) or determined by the FUID. For more information, see
                   Sub- DFC Function.

   Submit Print    Used to print reports or labels, or to reprint them. Inputs are populated based
   Request         on the report or label definition. For more information, see Submit Print
   User Formula    Request Function.

   Web Service     This type of Function is used to perform computations using JScript, Visual
                   Basic Script, or C# formulas. For more information, see User Formula
                   Function.

                   The Function invokes remote Web Service using the URL of an WSDL
                   document. For more information, see Web Service Function.
