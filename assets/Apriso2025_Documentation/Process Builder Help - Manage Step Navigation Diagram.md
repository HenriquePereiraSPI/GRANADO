# Manage Step Navigation Diagram

Perform an Auto Layout
   Choose Autolayout from the Right-Click Menu or use Ctrl+Shift+A Keyboard Shortcuts.
   The system will execute the automatic Auto Layout of the Step Navigation diagram.
   Print the Diagram
   Use one of the following options:

              File > Print from the Main Menu Bar
              Use the Ctrl+P Keyboard Shortcut
   A standard Windows printing dialog box will be displayed.

  It is possible to preview the Step Navigation diagram content before printing by using the File
  > Print Preview option.

Export the Diagram as an Image

Use one of the following options:

           Select Navigation > Save As Image... from the Main Menu Bar
           Use the F12 Keyboard Shortcut

A standard Save As dialog box will be displayed.

The available image types are: JPG, BMP, and GIF.

  Consider using the Autolayout feature before exporting the diagram.

Refresh the Diagram

Use one of the following options:

           Choose View > Refresh from the Main Menu
           Click (Refresh) on the Toolbar
           Use the F5 Keyboard Shortcut

The system will refresh the current view.
