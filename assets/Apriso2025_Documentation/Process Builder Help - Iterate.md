# Iterate

<htmltag data-flx-iterate="i,ListA,ListB"></htmltag>

"data-flx-iterate" duplicates the specific "htmltag" based on the parameters (e.g., <li> or <tr>)

Inputs                               HTML Layout Editor/HTML Editor                  Runtime
           DFC Function Navigation:

                                     HTML code

HTML View:

In this case, the style from the div tag is applied to every element from the list.

The Iterate converter cannot be used with the Complex data type.

Iterate with Children                HTML Layout Editor/HTML Editor                  Runtime

Inputs
           DFC Function Navigation:

                                     HTML code

HTML View:

  In this case, the style from the div tag is applied once for all of the elements from the list.
