# Add User Controls

Add a User Control

User Controls which can be added to Layout Editor from the Toolbox:

           Layout Controls (Grid (Deprecated) and Sub-DFC)
           Input Controls (see User Interface Controls for Function Input)
           Output Controls (see User Interface Controls for Function Output)
           Standard and Functional Business Controls (see Business Controls)

To Add a User Control to the Design Surface

Use one of the following options:

           Drag-and-drop the chosen User Control type from the Toolbox (see Available Elements in
           Layout Editor) onto the design surface
           When the User Control is not a grid, the system adds an appropriate Input or Output to
           dedicated Functions:
           For Inputs, the system creates the GetInputs Function (this Function is marked by the color
           yellow, and it is not possible to change the type of this Function)
           For Outputs, the SetOutputs Function is added (this Function is marked by the color
           yellow, and it is not possible to change the type of this Function)
           When a Business Control or Sub-DFC is added, the system creates a new Business
           Control Function or Sub-DFC Function within the Step

The user is able to add grids, Sub-DFCs and Business Controls in the same way as User Controls.

The name of a Sub-DFC and its revision is shown in the Layout Editor.
