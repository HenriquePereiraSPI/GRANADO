# Status Bar

The status bar offers the following information and functionalities:

Field           Description

Server          Indicates whether you are (Connected To) or (Disconnected From) a
Connection      server. When connected, the server name and your login name will be displayed.
Status

Sign off Status Indicates whether an electronic signature is required when working with
                       processes, operations, and DFCs. For details, refer to 21-CFR-Part-11
                       Compliance Guide.

Design Mode     Indicates whether the editing of entities is allowed. If the design mode is
Status          disabled, you cannot modify any entities. This can be set in the Central
                Configuration file, or when the entity is transferred between DELMIA Apriso
                servers.

Editability     Indicates whether an entity is (Editable) or (Read-only). The factors that
Indicator       influence this indicator are:

                   Entity status – entities are read-only in the active status.
                   Entity view – entities are read-only in Test Run.
                   Design mode status – if the design mode is disabled in the Central
                   Configuration file, entities are read-only.

Selected        Displays the number of selected items and the total number of filtered entities.
Entities Count
Entity Status   Indicates the status of the entity that is being edited. Right-click this part of the
                status bar to change the status of an entity.
Zoom            Indicates the current level of zoom at which the designer view is set. Right-click
                this part of the status bar to use pre-set zoom percentages.

                          This feature is disabled in the Test Run.
The status bar also displays information about installing updates and loading data.
