# Application Startup_Shutdown

Application Startup/Shutdown

To Start Up the Application Using the Non-Single Sign-On (SSO)
Route

Use one of the following options:

           Follow the path: Start > Programs > Dassault Systemes > DELMIA Apriso 2025
           Process Builder [SERVER NAME].
           Double-click the shortcut icon on your desktop.
           Run FlexNetProcessBuilder.exe from the command line.

The system uses the ClickOnce™ technology to verify that the latest version of the software is
present on the client machine, and makes the proper updates accordingly.

  Because of a third-party issue with Windows 11, Process Builder might not restart automatically
  after updates are installed or the ClickOnce is regenerated. Open the application manually.

The application's splash screen appears and the required initialization is performed. The main
console window and login dialog box are then displayed.

To Shut Down the Application

Use one of the following options:

           Exit from the File menu
           X in the upper-right corner of the main console window
            Alt+F4 shortcut

If you are working on an entity, you will be prompted to save it before leaving.
