# Modify the Properties of an Entity

Modify the Process or Operation Code

To Modify the Process Code

  The functionalities related to process authoring are deprecated and will not be further
  developed.

       1. Launch Entity Manager.
       2. Select the Process that you would like to modify the internal name of from the available

           list.
       3. Use one of the following options:

                      Choose Properties from the Right-Click Menu
                      Use the Shift+F4 Keyboard Shortcut
       4. Click the Edit... button next to the Process code field. This will display a dialog box used
           for editing the code.
       5. Enter the new name in the field provided in the dialog box. Click OK.
       6. Click OK again on the main Properties screen to confirm the change.

To Modify the Operation Code

       1. Duplicate the selected Operation to a new entity.
       2. Choose a new name and fill in the Entity name field.
       3. Change the original Operation status to Cancelled.

Operation Codes must be unique across Entity Manager or a single Project. The Codes are case-
sensitive and cannot start or end with a space character.
