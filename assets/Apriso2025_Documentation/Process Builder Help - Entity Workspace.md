# Entity Workspace

This is the main window where entities are displayed in the various view modes. Opened projects
or entities are displayed in a tabbed workspace with tool panes on both sides of the screen – you
can show or hide them using the View menu.

Workspace Tabs

When you have more than one entity opened, the tabs enable easier navigation between entities.
You can change the tab order by dragging the tab left or right. Use CTRL+TAB to switch between
tool panes or tabs.

Test Run

The Test Run view is available only for DFCs and screens. You can perform a test execution of a
designed DFC or screen without exiting Process Builder.

 Navigation Between Different Related Entities

  In Entity Explorer (depending on the assigned role):

              Use Backspace to navigate (items are highlighted going up in Entity Explorer) and Enter
              to open the selected view mode.
              Double-click to open an entity or its component.
              In the Sub-DFCs tab view:

                         Double-click to open a sub-DFC (a default action can be set when a DFC is
                         double-clicked in the Entity Explorer pane or on the Advanced tab of the Options
                         dialog box).
                         Right-click the menu of the sub-DFC function type, and a DFC can be edited in a
                         new tab or be displayed in the current one.

  On diagrams (depending on the assigned role):

              Double-click to open an entity or its component.
              In the Sub-DFCs tab view:

                         Double-click a step to open the sub-DFC view.
                         In the sub-DFC view, right-click for menu options or double-click to open the DFC
                         (a default action can be set when an DFC in this view is double-clicked on the
                         Advanced tab of the Options dialog box).

  If you are assigned the Process Technical Author role, the Open option is available on the sub-DFC
  function properties pane, displays the linked sub-DFC in a new tab.
