# Function Navigation Right-Click Menu

Function Navigation Workspace

This can be accessed by using the context menu keyboard key or by right-clicking anywhere in the
Function Navigation workspace.

                   Command     Description

                   Add         Adds a new function to the step. By
                   Function    default, the type of new function is Input
                               to Output.

                   Set Routing Enables setting a function output that will

                   Output      be used as an output routing of a DFC

                               step.

                   Cut         Cuts (the first part of the Move action)

                               the selected Function. The user can

                               select more than one Function to be cut.

                   Copy        Copies the selected Function. The user
                               can select more than one Function to be
                               copied.

                   Paste       Pastes the Function from the clipboard.
                               This is the second part of both the Cut
                               and Copy commands.

                   Delete      Deletes the selected Function.

                   Autolayout Automatically arranges all the items on
                                     the diagram.

                   Save As     Saves the flow diagram as an image. An
                   Image...    alternative method is to select entities
                               and press Ctrl+C. The selected
                               Functions will be copied to the clipboard
                               as an image (which can be pasted into
                               Microsoft Word or another application).

                   Properties Gives focus to the Properties Tool Pane
                                     for the selected Step.
