# Screen_ Roles Tab

Screen: Roles Tab

  On this tab, clicking (Add) invokes the Selecting Roles window, which lists all the Roles (for
  more information on defining Roles, refer to the Role Configuration topic in the Role and Skill
  Configuration Help ). The added Roles will have access to the Screen.
  If any employee does not have the required Role, an error message will be displayed with the
  information that the employee is not allowed to access the Screen.
  After selecting a Role, it appears in the field with its Name and Description.

     If no Roles are defined, the Screen will be accessible to everyone.
