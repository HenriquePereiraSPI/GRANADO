# Adjust Personalization Settings

You can adjust the following settings:

           Language Settings
           Time Zone Settings
           Test Run Language Settings

Language Settings

       1. Select Tools > Options... from the Main Menu.
       2. On the Localization tab, choose a language from those available in the Language drop-

           down list (for the list of supported languages, refer to Process of Translation Guide).
       3. Click OK to apply the changes.

 Time Zone Settings

          1. Select Tools > Options from the Main Menu.
          2. On the Localization tab, choose a zone from those available in the Time zone drop-down

              list.

          3. Click OK to apply.
     In order for the changes to take effect, the application needs to be restarted.

 Test Run Language Settings

          1. Select Tools > Options from the Main Menu.
          2. On the Localization tab, choose a language from those available in the Test Run

              Language and Test Run Database Language drop-down lists. For instructions on setting
              languages available on this list, refer to the Translations Settings topic in the Dictionary
              Help .

          3. Click OK to apply.
