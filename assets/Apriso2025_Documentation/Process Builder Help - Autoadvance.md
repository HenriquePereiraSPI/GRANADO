# Autoadvance

There are two ways to perform autoadvance:

           pressing the ENTER key
                      for the textarea element – press Shift+Enter keyboard shortcut ( Ctrl+Enter
                      shortcut for Screen Flows client mode)

           when the number of entered characters in control equals the maxlength value (not
           supported on a textarea element)

It is possible to set the tabindex attribute to -1 in order to skip a focus on the element.

<htmltag data-flx-autoadvance = "[true/false]" maxlength ="[number]"></htmltag>
