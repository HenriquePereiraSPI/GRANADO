# HTML View Type

This type of View consists of an HTML Editor, which is a simplified HTML Layout Editor, and an
Actions tab.
The HTML View is best suited for client mode and Offline Screens.
This View does not have:

           linked View DFC
           Forms tab

HTML View Properties

 HTML View Inputs

  This grid shows all the Inputs of the HTML View. The Inputs can be added in the grid by clicking
  (Add), or by typing in the name of the Input in curly brackets, e.g., {Input1}.

  The Inputs can only be deleted if they are not used in the HTML tab.

  $ScreenModel - this is an object that holds the context of the View/Screen in the JSON format. It is
  added to HTML View belonging to the Offline Module.

  Default Value is not editable for the Inputs of the Complex/List of Complex data type.

HTML View Outputs

This grid shows all the Outputs of the HTML View. The Outputs can be added in the grid or in the
HTML tab.
The Outputs can only be deleted if they are not used in the HTML tab.

Screen Literals

This pane shows all the literals defined for the HTML View.
For more information, see Screen Literals.

Mobile and Text platforms are no longer supported.
