# Duplicate User Controls

Duplicate a User Control

To Duplicate a User Control

       1. Select the User Control to be duplicated.
       2. Use one of the following options:

                      Choose Entity >New Revision/Duplicate from the Main Menu
                      Choose Duplicate Selection from the right-click menu
                      Use the Ctrl+D Keyboard Shortcut
       3. The system duplicates the selected User Controls:

                      If all items belong to the same grid, they are added at the end of that grid
                      Otherwise, the items are added at the bottom of the screen
       4. The system creates the following User Controls related to duplicated controls:

                      Function Inputs or Outputs – inside dedicated Functions
                      New Business Control Functions
                      New Sub-DFC Functions
