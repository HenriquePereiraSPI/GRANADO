# Manage Links

For details on adding and removing links, see Links.

   To Move a Link
           1. Use one of the following options:
                          Drag the link from the source location in the Links tab or Entity Explorer and
                          drop it in the target entity or step in Entity Explorer.
                          Select the link in the Links tab or Entity Explorer and:
                                     Use the Ctrl+X and Ctrl+V keyboard shortcuts.
                                     Choose Cut and Paste from the Edit menu.
                                     Click (Cut) and (Paste) in the toolbar.
                                     Choose Cut and Paste from the Entity Explorer right-click menu or
                                     the Links tab right-click menu.
           2. The link disappears from the source location and reappears in the target one.

   To Copy a Link
           1. Use one of the following options:
                          Hold the Ctrl key and drag the link from the Links tab or Entity Explorer and
                          drop it in the target entity or step in Entity Explorer.
                          Select the link in the Links tab or Entity Explorer and:
                                     Use the Ctrl+C and Ctrl+V keyboard shortcuts.
                                     Choose Copy and Paste from the Edit menu.
                                     Click (Copy) and (Paste) in the toolbar.
                                     Choose Copy and Paste from the Entity Explorer right-click menu or
                                     the Links tab right-click menu.
           2. A copy of the link (containing the same details) appears in the target location.

   Edit the Details of a Link

        Only the details of characteristics, resources, and components are editable.

              1. Use one of the following options:
                            Right-click the link in the:
                                        Links tab, and choose Details from the right-click menu.
                                        Entity Explorer, and choose Details from the right-click menu.

              2. An editable properties dialog box for the selected link appears. Modify the details, and
                 click OK to confirm.

      The Link Details screen opens in a read-only mode if the link was configured on an operation
      level. To edit it, access it from an operation context instead.

      The screen is editable for the links added from a process (even those added to linked operation).

      Preview a Document
              1. Use one of the following options:
                            When adding a document, you can select it and use the Preview option in the
                            Link Document Wizard.
                            Select a linked document from the Links tab and choose Preview from the
                            right-click menu.
              2. The document opens in the appropriate viewer, for example Adobe® Reader®.
