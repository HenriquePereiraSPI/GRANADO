# Searching for an Action Script

Search for an Action Script

It is possible to search within Action Script names and revisions as well as inside the Action Script
code by providing the text or the substring of text. If any of the searched text was found inside an
Action Script revision, it will be highlighted on the tree.

The right/left arrow buttons can be used to navigate to the next/previous matching node,
respectively. Navigation to the next matching node always results in the expansion of all the parent
nodes (and then only the matching node is marked).

To Find an Action Script or Action Script Code Element

       1. Open the Action Scripts Manager.
       2. Enter a text string into the Entity Explorer search field. The search results will be

           highlighted on the Entity Explorer tree.
       3. Use the right/left buttons to move through the search results.
