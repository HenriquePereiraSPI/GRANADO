# Screen Header

The Header can be displayed in the following environments:

           DELMIA Apriso Portal
           DELMIA Apriso Desktop Client

A Header is always displayed in the DELMIA Apriso Portal, so there is no need to link it to a screen.

The Header should not be linked to screens intended for use in DELMIA Apriso
3DEXPERIENCE Platform Widget.

For the DELMIA Apriso Desktop Client, the Header needs to be linked to the screen or linked to the
layout in order to be displayed.

Standard Header Functionalities

           Breadcrumbs
           Search (allows for searching only base screens with security)
           Standard buttons (Back, Home, Help)

                      The help button can be linked to custom documentation content prepared for the
                      created screen – for details, see Linking Custom Documentation Content to a
                      Screen
                      These can be enabled or disabled
                      Additionally, custom buttons can be added that call any action
           Clock
           User Information (clicking the portrait reveals the Logout button)
           Additional Information (can be added by the user by linking the Header View DFC)

For details on Header configuration refer to Header View Type.
