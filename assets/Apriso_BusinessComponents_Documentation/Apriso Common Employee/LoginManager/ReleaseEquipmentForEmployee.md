# ReleaseEquipmentForEmployee

**Category:** Apriso/Common/Employee
**Class:** `FlexNet.BusinessFacade.Security.LoginManager`
**Assembly:** `FlexNet.BusinessFacade.Security.dll`
**Status:** Active

## Description

This method is used to release a piece of equipment. It is released for use by a specific employee.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | EquipmentID | Integer | Yes | ID of the equipment which should be released. |
| Input | EmployeeID | Integer | Yes | ID of the employee which use equipment |

## System Processing

System releases a specific piece of equipment for a specific employee.
