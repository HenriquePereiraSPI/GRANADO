# ConvertBitToHex

**Category:** Apriso/Common/Code
**Class:** `FlexNet.BusinessFacade.Codes.CodeManager`
**Assembly:** `FlexNet.BusinessFacade.Codes.dll`
**Status:** Active

## Description

The purpose of this method is to convert bit code to its hexadecimal representation.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | bitCode | Char | Yes | Bit code to convert to. |
| Output | hexCode | Char | No | Hexadecimal code that will be returned. |

## Validations

- Entered code must belong to supported codes. 
- Entered code must be a correct bit number.

## System Processing

System converts input bit code to its hexadecimal representation.
