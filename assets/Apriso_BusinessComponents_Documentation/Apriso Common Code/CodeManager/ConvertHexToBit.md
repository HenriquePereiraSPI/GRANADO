# ConvertHexToBit

**Category:** Apriso/Common/Code
**Class:** `FlexNet.BusinessFacade.Codes.CodeManager`
**Assembly:** `FlexNet.BusinessFacade.Codes.dll`
**Status:** Active

## Description

The purpose of this method is to convert hexadecimal code to its bit representation.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | hexCode | Char | Yes | Hexadecimal code to convert to. |
| Output | bitCode | Char | No | Bit code that will be returned. |

## Validations

- Entered code must belong to supported codes. 
- Entered code must be a correct hexadecimal number.

## System Processing

System converts input hex code to its bit representation.
