# SetSignatureHeaderContext

**Category:** Apriso/Common/Signatures
**Class:** `FlexNet.BusinessFacade.Signature.SignatureManager`
**Assembly:** `FlexNet.BusinessFacade.Signature.dll`
**Status:** Active

## Description

This Business Component method sets the Signature Header Context with the values as defined by the inputted Signature Header Code.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | SignatureHeaderCode | Char | Yes | The Signature Header Code for which the Context will be set. |
| Input | SignatureHeaderFUID | Char | Yes | The Signature Header FUID for which the Context will be set. |

## System Processing

-  System sets the Signature Header Context values based on the inputted values.
