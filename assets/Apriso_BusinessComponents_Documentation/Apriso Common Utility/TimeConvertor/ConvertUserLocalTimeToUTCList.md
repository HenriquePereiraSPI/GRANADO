# ConvertUserLocalTimeToUTCList

**Category:** Apriso/Common/Utility
**Class:** `FlexNet.BusinessFacade.Utility.TimeConvertor`
**Assembly:** `FlexNet.BusinessFacade.Utility.dll`
**Status:** Active

## Description

This Business Component method converts the local time to UTC time based on the user's time zone. The method operates on lists.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | LocalTimeList | DateTime | Yes | The list of user's local time values to be converted. |
| Output | UTCTimeList | DateTime | No | The output list of time values in UTC. |
