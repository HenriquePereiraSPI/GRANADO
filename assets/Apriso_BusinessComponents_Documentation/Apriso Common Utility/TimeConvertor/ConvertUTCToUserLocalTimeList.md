# ConvertUTCToUserLocalTimeList

**Category:** Apriso/Common/Utility
**Class:** `FlexNet.BusinessFacade.Utility.TimeConvertor`
**Assembly:** `FlexNet.BusinessFacade.Utility.dll`
**Status:** Active

## Description

This Business Component method converts the UTC time to the local time based on the user's time zone. The method operates on lists.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | UTCTimeList | DateTime | Yes | The list of UTC time values to be converted. |
| Output | LocalTimeList | DateTime | No | The output list of time values in user's local time zone. |
