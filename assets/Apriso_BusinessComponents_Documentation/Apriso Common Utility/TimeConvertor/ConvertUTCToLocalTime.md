# ConvertUTCToLocalTime

**Category:** Apriso/Common/Utility
**Class:** `FlexNet.BusinessFacade.Utility.TimeConvertor`
**Assembly:** `FlexNet.BusinessFacade.Utility.dll`
**Status:** Active

## Description

This method converts the UTC time to local time based on the input time zone ID. This method can be used to get the local time in a time zone for a given UTC time.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | utcTime | DateTime | Yes | UTC Time to be converted. |
| Input | timeZoneID | Integer | Yes | Source Time Zone ID. |
| Output | localTime | DateTime | No | Converted local time output. |
