# ConvertUTCToUserLocalTime

**Category:** Apriso/Common/Utility
**Class:** `FlexNet.BusinessFacade.Utility.TimeConvertor`
**Assembly:** `FlexNet.BusinessFacade.Utility.dll`
**Status:** Active

## Description

This method converts the UTC time to local time based on the user time zone ID. This method can be used to get the local time at user time zone for a given UTC time.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | utcTime | DateTime | Yes | UTC Time to be converted. |
| Output | localTime | DateTime | No | Converted local time output |
