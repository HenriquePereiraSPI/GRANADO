# DecodeDynamicParameters

**Category:** Apriso/Common/Utility
**Class:** `FlexNet.FunctionInterpreter.BusinessFacade.DFCInterpreter`
**Assembly:** `FlexNet.FunctionInterpreter.BusinessFacade`
**Status:** Active

## Description

Decodes serialized parameters into Business Component Function Outputs. Outputs to decode needs to be added manually to the Business Component Function.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | EncodedParameters | Char | Yes | Encoded parameters, usually read from "DynamicOutputs" Sub-DFC Output. |

## System Processing

Decodes DynamicParameters Input into custom Business Component Outputs
