# DisplayMessage

**Category:** Apriso/Common/Utility
**Class:** `FlexNet.BusinessFacade.Utility.MessageGenerator`
**Assembly:** `FlexNet.BusinessFacade.Utility.dll`
**Status:** Retired

## Description

This method always returns success, which causes the message associated with the literal ID of the text to be displayed as output. This can be used to display localized messages.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | LiteralID | Char | Yes | Literal ID used to retrieve the message. |

## Pre-Conditions

Literal translations present in literal tables for the input literal ID.
