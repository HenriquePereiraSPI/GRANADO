# MachineUploadFileList

**Category:** Apriso/Common/Documentation
**Class:** `FlexNet.BusinessFacade.Codes.DocumentManager`
**Assembly:** `FlexNet.BusinessFacade.Codes`
**Status:** Active
**Keywords:** Machine, Upload

## Description

Returns a list of file names that exist in the Upload Source Path defined for an Equipment.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | equipmentID | Integer | Yes | The ID of the Equipment (from the EQUIPMENT table) |
| Output | xmlFileList | Char | No | The list of files located in the Upload Source Path for the Equipment |
