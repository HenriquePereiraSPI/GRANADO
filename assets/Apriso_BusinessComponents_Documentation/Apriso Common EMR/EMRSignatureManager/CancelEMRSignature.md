# CancelEMRSignature

**Category:** Apriso/Common/EMR
**Class:** `FlexNet.BusinessFacade.EMR.EMRSignatureManager`
**Assembly:** `FlexNet.BusinessFacade.EMR`
**Status:** Active
**Keywords:** CancelEMRSignature, Cancel, EMR, Signature

## Description

This Business Component method is used to delete an EMR header record and all of the associated EMR records and EMR field records with the specified EMR header ID or EMR header number.

## Parameters

| IO | Name | Type | Required | Description |
|----|------|------|----------|-------------|
| Input | EmrRecordID | Integer | Yes | The ID of the EMR record. |
| Input | SignatureDetailList | ListofInteger | Yes | The list of the signature detail IDs to be cancelled. |
| Input | EmployeeNo | Char | Yes | The employee number or login name of the employee who signed the signature. |
| Input | Password | Char | Yes | The employee's password. |
| Input | Comment | Char | No | The comment. |

## Validations

- The system validates if EmrRecordID exists in the system. 
- The system validates if each signature detail in the signature details list exists in the system.  
- The system validates EmployeeNo and the password.

## System Processing

- The system finds each signature from the signature detail list. 
- If the signature header has the Enforce Sequence field selected, the system will cancel not only the selected signature but also the signatures with the configured subsequent Sequence Numbers. 
- The system cancels each signature from the list.
