# UOM

**Database:** Operational

**Description:** The “UOM” table contains all standard uom codes.

## Columns

| Name | Data Type | Nullable | Default | Primary | FK To | Description |
|------|-----------|----------|---------|---------|-------|-------------|
| DecimalPlaces | INT(10,0) | True |  | False |  | Maximum number of decimal places. |
| RoundingMode | TINYINT(3,0) | True |  | False |  | 1-Truncate, 2-Ceiling, 3-Floor |
| TextID | INT(10,0) | True |  | False |  | Unique identifier of the Text. Can be used for translation purposes. |
| UnitOfMeasureBasis | NVARCHAR(10) | True |  | False | UOM_BASIS | Relation to base UOM |
| UnitOfMeasureSystem | NVARCHAR(10) | True |  | False |  | For future use. |
| UomCode | NVARCHAR(10) | False |  | True |  | Unit of Measure |
