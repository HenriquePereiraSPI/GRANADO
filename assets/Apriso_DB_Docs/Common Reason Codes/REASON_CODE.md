# REASON_CODE

**Database:** Operational

**Description:** This table contains the various Reason Codes that can be used in the system.

## Columns

| Name | Data Type | Nullable | Default | Primary | FK To | Description |
|------|-----------|----------|---------|---------|-------|-------------|
| Code | NVARCHAR(40) | True |  | False |  | The actual code (as the Reason Code field is unique). |
| Conforming | BIT | True |  | False |  | Indicates if the Reason Code is conforming (e.g., for quality status codes). |
| CostClassID | INT(10,0) | True |  | False | COST_CLASS | For future use. |
| Department | NVARCHAR(40) | True |  | False | DEPARTMENT | For future use. |
| DoubletimeGLID | INT(10,0) | True |  | False | GL_ACCOUNT | For future use. |
| Facility | NVARCHAR(40) | True |  | False | FACILITY | The assignment of the Reason Code to a Facility. |
| OvertimeGLID | INT(10,0) | True |  | False | GL_ACCOUNT | For future use. |
| ReasonClassID | INT(10,0) | True |  | False | REASON_CLASS | The link to the class. |
| ReasonCode | NVARCHAR(20) | False |  | True |  | The Reason Code |
| ReasonType | SMALLINT(5,0) | True |  | False | REASON_TYPE | The type of Reason Code. |
| RegularGLID | INT(10,0) | True |  | False | GL_ACCOUNT | The regular general ledger for this Reason Code. |
| TextID | INT(10,0) | True |  | False |  | Unique identifier of the Text. Can be used for translation purposes. |
| UnitID | INT(10,0) | True |  | False | UNIT | Unique identifier of the Unit. |
| Usage_ | SMALLINT(5,0) | True |  | False | REASON_CODE_USAGE | The usage of the Reason Code (quality, production, warehouse, machine time, etc.). |
